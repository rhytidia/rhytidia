## Rhytidia (she/her/hers)

I'm a _very_ amateur coder who is learning to code. Mostly python so far with a tiny bit of C thrown in, plus I can do a little bit of bash/zsh and basics of git. I've done some learning on boot.dev and also freecodecamp.org, and reading books like [_Automate the Boring Stuff with Python_](https://automatetheboringstuff.com/) by Al Sweigert and [_Beej's Guide to C Programming_](https://beej.us/guide/bgc/) by Brian “Beej Jorgensen” Hall.

I have been in academia for over 25 years in the humanities, where I still work (in Canada). I am learning coding partly so that I can contribute to open source projects in the future, and maybe even do some freelance work in retirement. Also just because I'm really enjoying it!

I will be using this repository to share projects I am working on as I learn. Many will be very simple, some more complex depending on what I'm learning. Constructive feedback on problems or things that could be better optimized or refactored are welcome!

## Currently working on

1. Working through the Memory Management course in C on boot.dev, and reading _Beej's Guide to C_ (linked above) to help.
2. Starting on a personal project in python (starting a repo for that soon)
